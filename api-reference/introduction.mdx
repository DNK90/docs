## 🚀 What is LedgerX?

**LedgerX** enables you to:

- Connect and **sync blockchain wallets** (e.g., Solana, Ethereum)
- Fetch **transactions and balances** across multiple blockchains
- Apply **accounting logic and PnL calculations**
- Export data for **tax and financial reporting**

The API is optimized for:

- High-throughput data ingestion
- Accurate fiat conversion
- Enterprise-grade data integrity

## 🔑 Authentication

All API endpoints require an `API_KEY` header:

```
curl -X GET "https://api.ledgerx.finance/v1/wallets" \
  -H "API_KEY: <your-api-key>"
```

You can generate an API Key in your **LedgerX Dashboard → API Settings**.

## 🧠 Design Philosophy

- **Simple but expressive:** RESTful endpoints with consistent structure
- **Predictable pagination:** always return `{ data, totalItems, totalPages }`
- **Secure by default:** every call requires authentication
- **Composable:** endpoints are modular and work across chains
